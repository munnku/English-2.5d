# 06. 商用利用の定義とツール料金まとめ

ゲーム開発においてAIツールを活用する場合、**「商用利用（Commercial Use）」のルールと料金体系**を理解しておくことが非常に重要です。

---

## 1. 「商用利用」の定義について
ゲーム開発や同人活動における「商用利用」とは、一般的に**「そのコンテンツを用いて直接的あるいは間接的に収益を得る活動」**すべてを指します。

### 具体的に商用利用とみなされる例
*   **ゲームの有料販売:** Steam、App Store、Google Play、BOOTHなどでのパッケージ・ダウンロード販売。
*   **ゲーム内課金の導入:** ガチャやアイテム、追加DLCなどの販売。
*   **無料ゲームでの広告収入:** スマホアプリ内のバナー広告、またはブラウザゲーム公開サイトでの広告収益。
*   **投げ銭・寄付:** YouTubeのスーパーチャットやプレイ動画での収益化、Patreon、クリエイター支援サイトなどでの資金調達のための利用。
*   **企業・法人としての利用:** たとえ無料のプロモーションゲームであっても、法人が制作する場合は商用利用とみなされます。

**※注意点**
「趣味で作って無料で公開するだけ」なら非商用（無料プランでOK）ですが、「無料で公開するけど、アプリ内に広告を付けて小遣い稼ぎをしたい」となると**商用利用**になり、有料プランの契約が必要になるツールが多いです。

---

## 2. 各分野のAIツールの商用利用料金（2026年時点）

※料金は月額（Month-to-Month）の目安です。年額一括払いの場合さらに安くなる傾向があります。為替レートやプラン改定により変動する点にご注意ください。

### 🖼️ 2Dグラフィック・画像生成
商用利用するには、原則として有料プランの契約または自前のPCでの生成（ローカル生成）が必要です。

| ツール名 | 商用利用の条件 | 最安の商用可能プラン |
| :--- | :--- | :--- |
| **Midjourney** | 有料プランのみ | **Basic Plan:** 約$10/月 |
| **Leonardo.ai** | 有料プランのみ | **Apprentice:** 約$12/月 |
| **Stable Diffusion** | オープンソースにつき無料 | 無料（※利用するモデル（Checkpoint）ごとの利用規約に依存します）|

### 🧊 3Dモデル生成
3Dモデル界隈は、無料版だと「自分の作品を公開（Public）にしなければならない」などの制約がつくことが多いです。

| ツール名 | 商用利用の条件 | 最安の商用可能プラン |
| :--- | :--- | :--- |
| **Tripo AI** | 有料プランのみ | **Professional:** 約$20/月 ($19.90)|
| **Meshy AI** | **無料でも商用可能**（ただし「CC BY 4.0」につき、ゲーム内で「Meshy AIを使った」というクレジット表記が必須） | **Pro Plan:** $20/月（Proにすればクレジット表記不要になり私的権利を持てる）|

### 🎵 サウンド・音声生成
音声や音楽のAIツールは著作権がシビアなため、商用利用には「クレジット表記」か「少額の課金」が求められます。

| ツール名 | 商用利用の条件 | 最安の商用可能プラン |
| :--- | :--- | :--- |
| **Suno AI** (BGM) | 有料プランのみ | **Pro Plan:** 約$8〜$10/月 |
| **Udio** (BGM) | **無料でも商用可能**（ただし「Created with Udio」のクレジット表記必須） | **Standard Plan:** $10/月（クレジット表記不要になる） |
| **Soundraw** (BGM)| 有料プランのみ（ダウンロードには課金必須） | **Creator Plan:** 約$11〜$17/月 |
| **ElevenLabs** (SE/ボイス) | 有料プランのみ。無料版は非商用限定。 | **Starter Plan:** 約$5/月 |
| **VOICEVOX** (ボイス) | **完全無料・商用可能** | 無料（※ただしキャラクターごとに「ずんだもん」などのクレジット表記が必須）|

## 💡 初心者へのおすすめ戦略
開発初期（学習段階）は、「すべて無料プラン（非商用）」でツールを使い倒してプロトタイプを作りましょう。
そして「いざゲームが完成し、広告を付けたり販売したりしよう！」と決まった段階で、**必要なツールの最低限の有料プラン（$5〜$20程度）に1ヶ月だけ課金し、素材を商用ライセンス付きで生成し直す・ダウンロードする**というのが、最もコストパフォーマンスの良い賢い立ち回り方です。
