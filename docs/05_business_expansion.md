# ビジネス展開のロードマップと資金・投資戦略（日本国内・学生起業）

学生からスタートし、教育系SaaS/ゲームとして事業を拡大していくための詳細なシナリオと投資タイミングです。

## 1. 資金調達（日本特有の学生向け支援・助成金）
自己資金を使わずとも、日本には学生起業家を支援する手厚い制度が多数あります。これらを活用することがリスクゼロで拡大する秘訣です。
*   **大学の起業支援プログラム・ファンド**: 東京工業大学（最大100万円）や東京大学など、多くの大学が独自のスタートアップ支援金やインキュベーション施設を提供しています。自身が所属する大学の制度をまず確認してください。
*   **AIBC学生起業支援助成金など**: 民間団体が高校生・大学生向けに数十万円の助成金（返済不要）を出しているケースがあります。
*   **経産省・文科省のプログラム**: 経産省の社会起業家アクセラレーションプログラム「ゼロイチ」や、文科省の「START」など、教育課題の解決を目指すプロジェクトは国からの採択（メンタリングや資金援助）を受けやすい傾向にあります。
*   **日本政策金融公庫の創業融資**: いよいよ本格的に法人化して広告を打つフェーズになった場合、学生や若者向けの低金利・無担保融資制度が利用できます。

## 2. 事業規模別：展開の流れと投資タイミング

### Phase 1: MVP（最小限のプロダクト）開発・検証期 (Month 0-3)
*   **目標**: アイデアが仮説通りに機能するか（子供が本当に遊ぶか）を検証する。
*   **お金の投資**: **0円〜月数千円（限界まで抑える）**
    *   ドメイン取得: 年間約2,000円
    *   Vercel / Supabase: 無料Tireを利用
*   **時間の投資**: 全力でコーディング。泥臭く知り合いの親御さんに頼んで、子供の手元（ユーザーテスト）を観察させてもらいます。

### Phase 2: ソフトローンチ・初期トラクション獲得 (Month 3-6)
*   **目標**: 最初の100人のユーザーを獲得し、数人でもお金（サブスク）を払ってくれるかを確認する。
*   **お金の投資**: **月1万〜3万円**
    *   特商法対策用バーチャルオフィス: 月2,000円〜
    *   アセット（プロのイラストやBGM素材）：一括数千円〜数万円
    *   サーバー費用（Vercel Pro等）: ユーザーが増えてきたら月額$20程度。
*   **アクション**: SNS（XやInstagram）での開発過程の発信、テストユーザーへの継続的なアプローチ。

### Phase 3: プロダクトマーケットフィット（PMF）到達・グロース期 (Month 6-12)
*   **目標**: 収益モデル（LTV > CAC）が証明され、資金を投下して一気にユーザーを増やす。
*   **お金の投資**: **月5万〜数十万円（調達した助成金や利益を再投資）**
    *   広告費（Instagram / TikTok 等のSNS広告）
    *   コンテンツ制作外注費（新しい英語の問題の作成や、ネイティブによる音声吹き込みを外部ライターに依頼）
*   **アクション**: LTV（顧客生涯価値：ユーザー1人が解約するまでに払う総額）が、CAC（顧客獲得単価：1人を獲得するのにかかった広告費）を上回っている限り、広告を踏み続けます。

### Phase 4: B2B展開と法人化 (Year 1〜)
*   **目標**: 事業の安定化と社会的信用の獲得。
*   **投資**: 法人設立費用（株式会社設立に約25万円）。
*   **アクション**: 個人（B2C）だけでなく、**学習塾や学童保育向けの教材パッケージ（B2B）** としての営業を開始します。「塾での自習時間に子供が勝手にやってくれるツール」として、一括契約（数十ライセンスごと）を取りに行くことで、収益の柱が一気に太くなります。
